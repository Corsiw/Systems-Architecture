# Содержание репозитория
Домашки с курса **Архитектура компьютера и ОС** 2 курса ПИ Вышки.
* [DZ_1](DZ_1) - [DZ_5](DZ_5) - программы на асеммблере архитектуры RISC-V. Некоторые содержат отчеты, что, зачем и как.
* [IDZ_1](IDZ_1) - RICS-V асемблер, обработка массивов, работа со стеком, работа с подпрограммами и макросами.
* [IDZ_2](IDZ_2) - тоже самое, только работаем с **double** и используем численный метод хорд для решения уравнения. 
* [DZ_6](DZ_6) - базовая работа с Bash, вызов программ с аргументами, вызов другим скриптов и т.д.
* [DZ_7](DZ_7) - прога на **С** для копирования файлов в том числе и их разрешений.
* [IDZ_3](IDZ_3) - 4 версии программы на **C++**, работающей с **процессами**. Родственные процессы, POSIX, общая память, observer.
