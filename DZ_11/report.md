# **Отчёт по работе: Максимальная глубина рекурсии символьных ссылок** Терехов Дмитрий 10 баллов.

## Идея решения

1. Создаём отдельный каталог (например, `files`).
2. В нём создаём обычный файл `a`.
3. Далее в цикле:
   * создаём символьную ссылку `a1 - a`
   * пробуем `open("a1")`
   * если успешно — создаём `a2 - a1`, затем `a3 - a2` и т.д.
4. Как только `open()` **возвращает ошибку** (обычно `ELOOP` — слишком много символьных ссылок),
   текущее количество ссылок и есть **глубина рекурсии**.
5. Выводим результат.

---

## Ожидаемый результат

На моем Arch-linux результат следующий:

```
Ошибка открытия на глубине 42
errno = 40 (Too many levels of symbolic links)
Максимальная глубина рекурсии символьных ссылок: 41
```
